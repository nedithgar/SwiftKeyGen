# count

**Instance Property**

> The number of elements in the span.

**Availability:**
* iOS 12.2+
* iPadOS 12.2+
* Mac Catalyst 12.2+
* macOS 10.14.4+
* tvOS 12.2+
* visionOS 1.0+
* watchOS 5.2+

```swift
var count: Int { get }
```

## Discussion

To check whether the span is empty, use its isEmpty property instead of comparing count to zero.
